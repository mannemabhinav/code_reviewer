🤖 AI Code Reviewer
📌 Introduction

AI Code Reviewer is a Python-based application that analyzes Python code and provides feedback on code quality. It helps developers identify coding issues, improve readability, and follow coding standards.

🎯 Objective

To automate the code review process by checking code style, complexity, and formatting, while providing improvement suggestions.

🛠️ Tools Used
Python
Streamlit
Flake8
Radon
Black
Pandas
✨ Features
Upload Python files
Check coding style using Flake8
Measure code complexity using Radon
Auto-format code using Black
Generate code quality score
Provide improvement suggestions
Download analysis report
▶️ How to Run

Install dependencies:

pip install streamlit pandas flake8 radon black

Run the application:

streamlit run app.py
